Calculadora Python

Projeto desenvolvido para a formação de Analista de Dados da EBAC.

Objetivo

Criar uma calculadora simples utilizando:

* Input de dados do usuário
* Conversão para números
* Operações matemáticas
* Estruturas condicionais e/ou laços de repetição
* Exibição de resultados

Tecnologias utilizadas

* Python
* Google Colab
* GitHub
